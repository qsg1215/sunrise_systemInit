# 项目说明

## 项目启动步骤

```bash
npm install
```

或者

```bash
yarn
```

### 启动项目

```bash
npm start

npm run start //启动本地dev环境
npm run start:test //启动测试环境
npm run start:pre //启动预上线环境
npm run analyze //分析构建包的大小
 

```

### 构建项目

```bash
npm run build
```

### 代码检查

```bash
npm run lint
```

代码修复

```bash
npm run lint:fix
```

### 代码测试

```bash
npm test
```
